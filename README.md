# WebApp

    nodemon is recommended:

    npm install nodemon -g

## Server:

Terminal 1:

    git --version  

    node -v

    git clone https://github.com/WasteNotCSULB/WebApp.git

    cd WebApp/

    cd server/
        
    npm install 
    
    node server.js 
    
    ctrl + C
    
    nodemon
    
Browser: 

    go to http://localhost:3030    
    

## Client

Terminal 2:

    cd client/WasteNotCSULB

    npm install
    
    ng serve

Browser: 

    go to http://localhost:4200

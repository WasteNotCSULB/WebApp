# WebApp

    nodemon is recommended:

    npm install nodemon -g

## Server:

Terminal 1:

    git --version  

    node -v

    git clone https://github.com/WasteNotCSULB/WebApp.git

    cd WebApp/

    cd server/
        
    npm install 
    
    node server.js 
    
    ctrl + C
    
    nodemon

Terminal 2:

    cd client/
    cd WasteNotCSULB/

    npm install
    
    ng serve --open
